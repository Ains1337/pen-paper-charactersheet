# Pen-Paper-CharacterSheet

## Why

This project makes it possible to create multiple Characters with different settings like properties for strength, intelligence and skills like magic skills, fireball, or martial skills, poison arrow, including a search function to get quickly access to the different characters, properties and skills.
This project contains an area for players and game masters.

## How

Frontend: Bun, Solidjs mit Vite to create a lightweight GUI with input fields and buttons

Backend: GOLANG, API

DB: PostgreSQL

Availability: Cross-Plattform
